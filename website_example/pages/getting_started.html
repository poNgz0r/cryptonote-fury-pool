Start mining in 6 steps

  1. Download 0% fee miner: piratepools-xmr-stak-2.10.4.zip
  2. Extract zip file
  3. Open pools.txt
    4. "pool_list" :
[
	{
		"pool_address" : "POOL_HOST:PORT", 
		"wallet_address" : "WALLET_ADDRESS", 
		"rig_id" : "myRig", 
		"pool_password" : "myRig", 
		"use_nicehash" : false, 
		"use_tls" : false, 
		"tls_fingerprint" : "", 
		"pool_weight" : 1 
	},
],
  4. Change the PORT in the "pool_address" section to the port that matches your hashrate
  5. Change the WALLET_ADDRESS in the "wallet_address" section to your wallet address
  6. Run start.bat

Optional

If you want to disable AMD cards or Nvidia cards or want to disable CPU mining edit the start.bat
go to line 8 where you see xmr-stak and edit like this:
  • No AMD cards: xmr-stak --noAMD
  • No Nvidia cards: xmr-stak --noNVIDIA
  • No CPU mining: xmr-stak --noCPU
  • No AMD and Nvidia card: xmr-stak --noAMD --noNvidia
